Challenge 1 (Multiples of 3 or 5)

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.
Find the sum of all the multiples of 3 or 5 below the input parameter value number.





  1. Natural Numbers are counting numbers from 1 to infinity.
    To ensure all the numbers are tested before reaching input value, for statement is used.
    The initial value is set to 1, the condition is true until reaching input value, then the initial value is incremented by 1.
  2. Multiples of Numbers can be determined when products are formed from a certain number multiplies with an integer.
    This condition statement input%number === 0 is required.
    Whenever the remainder of modulo operation is strictly equal to 0, the input is indeed the multiplies of the number.
  3. Sum is an addition of all numbers.
    Since it involves several numbers to be added, an array is initialized with empty array.
    A variable for sum is initialized with 0, so that there is a value for the first index of the array to be added.
    To add all the numbers that are listed in array, for statement is used.
    The initial value is set to 0 (indicates the first index of the array), the condition is true until reached all indexes, then the initial value is incremented by 1.
    The sum calculation mySum += myArray[index] is nested within for statement.
