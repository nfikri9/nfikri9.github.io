CSS BOX MODEL

ELEMENT DIMENSIONS

Related Properties:

  • height: myValue;
    • The height property sets the height of an element
    • Related Value(s):
      • CSS Global Values
      • CSS Length Values
      • auto ( A default value where the element will automatically adjust its height to allow its content to be displayed correctly )

  • width: myValue;
    • The width property sets the width of an element
    • Related Value(s):
      • CSS Global Values
      • CSS Length Values
      • auto ( A default value where the element will automatically adjust its width based on display property's default value )

ELEMENT SPACING

Related Properties

  • padding: myValues;
    • The padding property specifies space around an element's content, inside of any defined borders
    • There are other properties related to padding property that can be written as its values
      • 4 values = padding: padding-top padding-right padding-bottom padding-left;
      • 3 values = padding: padding-top padding-Horizontal padding-bottom;
      • 2 values = padding: padding-Vertical padding-Horizontal;
      • 1 value = padding: padding-All;
    • Related Value(s):
      • CSS Global Values
      • CSS Length Values
      • 0 ( A default value where there is no padding )

  • margin: myValues;
    • The margin property specifies space around elements, outside of any defined borders
    • There are other properties related to margin property that can be written as its values
      • 4 values = margin: margin-top margin-right margin-bottom margin-left;
      • 3 values = margin: margin-top margin-Horizontal margin-bottom;
      • 2 values = margin: margin-Vertical margin-Horizontal;
      • 1 value = margin: margin-All;
    • Related Value(s):
      • CSS Global Values
      • CSS Length Values
      • 0 ( A default value where there is no margin specified )
      • auto ( Horizontally center the element within its container )

  • border: border-width border-style border-color;
    border-top: border-top-width border-top-style border-top-color;
    border-right: border-right-width border-right-style border-right-color;
    border-bottom: border-bottom-width border-bottom-style border-bottom-color;
    border-left: border-left-width border-left-style border-left-color;
    • border-width: myValues;
      • The border-width property specifies the width of an element's borders
      • There are other properties related to border-width property that can be written as its values
        • 4 values = border-width: border-top-width border-right-width border-bottom-width border-left-width;
        • 3 values = border-width: border-top-width border-Horizontal-width border-bottom-width;
        • 2 values = border-width: border-Vertical-width border-Horizontal-width;
        • 1 value = border-width: border-All-width;
      • Related Value(s):
        • CSS Global Values
        • CSS Length Values
        • medium ( A default value where the width of the border is medium )

    • border-style: myValues;
      • The border-style property specifies the style of an element's borders
      • There are other properties related to border-style property that can be written as its values
        • 4 values = border-style: border-top-style border-right-style border-bottom-style border-left-style;
        • 3 values = border-style: border-top-style border-Horizontal-style border-bottom-style;
        • 2 values = border-style: border-Vertical-style border-Horizontal-style;
        • 1 value = border-style: border-All-style;
      • Related Value(s)
        • CSS Global Values
        • none ( A default value where there is no specified border style )
        • solid ( The style of the border is solid )

    • border-color: myValues;
      • The border-color property specifies the color of an element's borders
      • There are other properties related to border-color property that can be written as its values
        • 4 values = border-color: border-top-color border-right-color border-bottom-color border-left-color;
        • 3 values = border-color: border-top-color border-Horizontal-color border-bottom-color;
        • 2 values = border-color: border-Vertical-color border-Horizontal-color;
        • 1 value = border-color: border-All-color;
      • Related Value(s):
        • CSS Global Values
        • CSS Color Values
        • currentColor ( A default value where the border color is based on the value of the color property of the element )

  • border-radius: myValues;
    • The border-radius property defines the radius of the element's corners
    • There are other properties related to border-radius property that can be written as its values
      • 4 values = border-radius: border-top-left-radius border-top-right-radius border-bottom-right-radius border-bottom-left-radius;
      • 3 values = border-radius: border-top-left-radius border-Diagonal-Top-Bottom-radius border-bottom-right-radius;
      • 2 values = border-radius: border-Diagonal-Left-Right-radius border-Diagonal-Right-Left-radius;
      • 1 value = border-radius: border-All-radius;
    • Related Value(s):
      • CSS Global Values
      • CSS Length Values
      • 0 ( A default value where there is no border radius specified )

MISC

Related Properties

  • outline: outline-width outline-style outline-color;
    • outline-width: myValue;
      • The outline-width property specifies the width of an outline, drawn around elements, outside the borders
      • Related Value(s):
        • CSS Global Values
        • CSS Length Values
        • medium ( A default value where the width of the outline is medium )

    • outline-style: myValue;
      • The outline-style property specifies the style of an outline, drawn around elements, outside the borders
      • Related Value(s):
        • CSS Global Values
        • none ( A default value where there is no specified outline )
        • solid ( Specifies a solid outline )

    • outline-color: myValue;
      • The outline-color property specifies the color of an outline, drawn around elements, outside the borders
      • Related Value(s):
        • CSS Global Values
        • CSS Color Values
        • currentColor ( A default value where the outline color is based on the value of the color property of the element )

  • outline-offset: myValue;
    • The outline-offset property adds space between the outline and the edge or border of an element
    • Related Value(s):
      • CSS Global Values
      • CSS Length Values
      • 0 ( A default value where there is no specified outline offset )

  • box-shadow: H-Offset V-Offset Blur-Radius Spread-Radius Color inset;
    • The box-shadow property specifies one or multiple shadows to an element
    • H-Offset
      • Required parameter which specifies the horizontal offset of the shadow
      • A positive value puts the shadow on the right side of the box
      • A negative value puts the shadow on the left side of the box
      • Related Value(s):
        • CSS Global Values
        • CSS Length Values
        • none ( A default value where there is no shadow )
    • V-Offset
      • Required parameter which specifies the vertical offset of the shadow
      • A positive value puts the shadow below the box
      • A negative value puts the shadow above the box
      • Related Value(s):
        • CSS Global Values
        • CSS Length Values
        • none ( A default value where there is no shadow )
    • Blur-Radius
      • Optional parameter which specifies the blur radius of the shadow
      • The higher the number, the more blurred the shadow will be
      • Related Value(s):
        • CSS Global Values
        • CSS Length Values
        • 0 ( A default value where there is no blur radius specified )
    • Spread-Radius
      • Optional parameter which specifies the spread radius of the shadow
      • A positive value increases the size of the spread radius shadow
      • A negative value decreases the size of the spread radius shadow
      • Related Value(s):
        • CSS Global Values
        • CSS Length Values
        • 0 ( A default value where there is no spread radius specified )
    • Color
      • Optional parameter which specifies the color of the shadow
      • Related Value(s):
        • CSS Global Values
        • CSS Color Values
        • currentColor ( A default value where the shadow color is based on the value of the color property of the element )
    • inset
      • Optional value which changes the shadow from an outer shadow to an inner shadow

INSIGHTS:

  • Requires atleast one property related to box dimension
    • If the height of an element is specified but the width of the element is omitted, then the width is automatically set to its default value, and vice versa
  • Actual dimensions
    • Height Dimension = "border-top-width" + "padding-top" + "height" + "padding-bottom" + "border-bottom-width"
    • Width Dimension = "border-left-width" + "padding-left" + "width" + "padding-right" + "border-right-width"
    • Property like outline does not affect element's dimensions and may overlap other content
  • Overflowed contents
    • If the content does not fit within the specified width, the contents will be displayed on the next line ( when the height property is specified ), or it will automatically change the height of the element ( when height: auto; )
    • If the content does not fit within the specified height, it will overflow
    • Use overflow property to manage the overflowed contents
  • Margin collapse
    • The margin collapse happens due to top and bottom margins collapsed into a single margin that is equal to the largest of those two margins
    • The margin collapse only happens on vertical margin (top and bottom), this does not happen on horizontal margin (left and right)
    • Example, if Element A has margin-bottom: 20px; and Element B has margin-top: 30px; the vertical margin between Element A and Element B would be 30px
    • Instead of being 50px ( 20px from Element A's bottom margin + 30px from Element B's top margin), the actual vertical margin is 30px due to margin collapse

Related Properties:

  • position: myValue;
    • The position property specifies the type of positioning method used for an element
    • Related Value(s):
      • CSS Global Values
      • static ( A default value which the elements render in order, as they appear in the document flow )
      • absolute ( The element is positioned relative to its first positioned (not static) ancestor element )
      • fixed ( The element is positioned relative to the browser window )
      • relative ( The element is positioned relative to its normal position )
      • sticky ( The element is positioned based on the user's scroll position )
