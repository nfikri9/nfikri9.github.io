HTML INTRODUCTION

What is HTML?

HTML (Hypertext Markup Language) is the standard markup language for creating Web pages
HTML describes the structure of a Web page
HTML consists of a series of elements
HTML elements tell the browser how to display the content

HTML Syntax

An HTML element is defined by an open-tag (<>), some contents, and a closed-tag (</>)
Example → <TagName> MyContents </TagName>
Some HTML elements have no content (like the <br> element)
These elements are called empty elements which do not have a closed-tag

<!DOCTYPE>

All HTML documents must start with a <!DOCTYPE> declaration
The declaration is not an HTML tag, but it is an "information" to the browser about what document type to expect
Therefore, this <!DOCTYPE html> declaration defines that this is an HTML 5 document (latest version of HTML)

INSIGHTS

HTML METADATA

<title> My Text </title>

The <title> tag defines the title of the document
The title must be text-only, and it is shown in the browser's title bar or in the page's tab

Related Attribute(s):

  • HTML Global Attributes
  • HTML Event Attributes
  • HTML DOM

Default CSS Setting(s):
title { display: none; }

INSIGHTS:

  • Standard structure for <title> element
    • The <title> element is placed within <head> tag
    • There can only be one <title> element in an HTML document

  • Improve SEO with <title> element
    • The contents of a page title is very important for search engine optimization (SEO)
    • The page title is used by search engine algorithms to decide the order when listing pages in search results

  • Best practice to create good title
    • Go for a longer, descriptive title (avoid one- or two-word titles)
    • Search engines will display about 50-60 characters of the title, so try not to have titles longer than that
    • Do not use just a list of words as the title (this may reduce the page's position in search results)

<link>

The <link> tag defines the relationship between the current document and an external resource

Related Attribute(s):

  • HTML Global Attributes
  • HTML Event Attributes
  • HTML DOM
  • rel="myValue"
    • Specifies the relationship between the current document and the linked document
    • Related Value(s):
      • icon
      • stylesheet

  • href="myValue"
    • Specifies the URL of the linked document
    • Related Value(s):
      • Absolute URL
        • Points to another web site
        • Example, href="http://www.example.com"
      • Relative URL
        • Points to a page within a web site
        • Example, href="example.html"

  • type="myValue"
    • Specifies the media type of the linked document
    • Related Value(s):
      • image/x-icon

Default CSS Setting(s):
link { display: none; }

INSIGHTS:

  • Standard structure for <link> element
    • The <link> element is placed within <head> tag

  • Exporting CSS file
    • Linking HTML document to external source of CSS file
    • Example, <link rel="stylesheet" href="myValue">

  • Creating favicon
    • A favicon is a small image displayed next to the page title in the browser tab
    • Example, <link rel="icon" type="image/x-icon" href="myValue">

<time> My Text </time>

The <time> tag defines a specific time (or datetime)

Related Attribute(s):

  • HTML Global Attributes
  • HTML Event Attributes
  • HTML DOM
  • datetime="myValue"
    • The datetime attribute represent a machine-readable format of a <time> element
    • Related Value(s):
      • YYYY-MM-DDThh:mm:ssTZD
        • YYYY stands for year ( Example, 2009 )
        • MM stands for month ( Example, 01 for January )
        • DD stands for day of the month ( Example, 08 )
        • T or a "space" stands for a separator, ( Required if time is also specified )
        • hh stands for hour ( Example, 22 for 10.00pm )
        • mm stands for minutes ( Example, 55 )
        • ss stands for seconds ( Example, 03)
        • TZD stands for Time Zone Designator ( Z denotes Zulu, also known as Greenwich Mean Time )
      • PTDHMS
        • P is a prefix for "Period"
        • T stands for a separator ( Like "space" )
        • D is a prefix for "Days"
        • H is a prefix for "Hours"
        • M is a prefix for "Minutes"
        • S is a prefix for "Seconds"

Default CSS Setting(s):
NONE
