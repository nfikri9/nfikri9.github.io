HTML FUNDAMENTAL STRUCTURES

<!DOCTYPE>

All HTML documents must start with a <!DOCTYPE> declaration
The declaration is not an HTML tag, but it is an "information" to the browser about what document type to expect
Therefore, this <!DOCTYPE html> declaration defines that this is an HTML 5 document

NOTES:

  • The <!DOCTYPE> declaration is NOT case sensitive

<html> My Contents </html>

The <html> tag represents the root of an HTML document
The <html> tag is the container for all other HTML elements, except for the <!DOCTYPE> declaration

Related Attributes:

  • HTML Global Attributes
  • HTML DOM
  • xmlns = "My Value"
    • The xmlns attribute specifies the XML namespace for a document
    • Related Values:
      • http://www.w3.org/1999/xhtml

Default CSS Settings
html { display: block; }
html:focus { outline: none; }

NOTES:

  • Best practice to include lang attribute
    • Always include the lang attribute inside the <html> tag, to declare the language of the Web page
    • This is meant to assist search engines and browsers

HTML SEMANTIC ELEMENTS (TEXT)

<!-- My Text -->

The comment tag is used to insert comments in the source code
Comments are not displayed in the browsers
Developers can use comments to explain their code, which can help them when they edit the source code at a later date
This is especially useful if they have a lot of code

<h1> My Text </h1>

The <h1> to <h6> tags are used to define HTML headings
The <h1> tag defines the most important heading while the <h6> tag defines the least important heading

Related Attribute(s):

  • HTML Global Attributes
  • HTML Event Attributes
  • HTML DOM

Default CSS Setting(s):
h1 { display: block; margin: 0.67em 0; font-size: 2em; font-weight: bold; }
h2 { display: block; margin: 0.83em 0; font-size: 1.5em; font-weight: bold; }
h3 { display: block; margin: 1em 0; font-size: 1.17em; font-weight: bold; }
h4 { display: block; margin: 1.33em 0; font-size: 1em; font-weight: bold; }
h5 { display: block; margin: 1.67em 0; font-size: .83em; font-weight: bold; }
h6 { display: block; margin: 2.33em 0; font-size: .67em; font-weight: bold; }

INSIGHTS:

  • The <h1> tag usage
    • It is recommended to use only one <h1> tag per page
    • This should represent the main heading/subject for the whole page
  • The flow of heading levels
    • Do not skip heading levels, start with <h1>, then use <h2>, and so on

HTML GLOBAL ATTRIBUTES

class Attribute

The class attribute specifies one or more class names for an element

Related Value(s):

  • myClass
    • Specifies one or more class names for an element
    • The class names must begin with a letter A-Z or a-z
    • The class names can be followed by: letters (A-Za-z), digits (0-9), hyphens ("-"), and underscores ("_")

INSIGHTS:

  • Syntax of writing multiple classes for an element
    • To specify multiple classes, separate the class names with a space
    • This allows developers to combine several CSS classes for one HTML element
    • Example, <div class="myClass1 myClass2 myClass3">
  • Mostly used as node
    • The class attribute is mostly used to point to a class in a style sheet
    • It can also be used by a JavaScript (via the HTML DOM) to make changes to HTML elements with a specified class

id Attribute

The id attribute specifies a unique id for an HTML element

Related Value(s):

  • myId
    • Specifies a unique id for the element
    • The id name must contain at least one character
    • The id name must not contain any space characters

INSIGHTS:

  • Unique Id
    • The value must be unique within the HTML document, which only one specific id can exist in the document
  • Mostly used as node
    • The id attribute is mostly used to point to a style in a style sheet
    • It can also be used by a JavaScript (via the HTML DOM) to manipulate the element with the specific id

lang Attribute

The lang attribute specifies the language of the element's content

Related Value(s):

  • HTML Language Codes
    • Specifies the language code for the element's content
    • Common examples are "en" for English, "fr" for French, and so on

style Attribute

The style attribute specifies an inline style for an element

Related Value(s):

  • CSS Properties
    • Defines an inline style for an element

INSIGHTS:

  • Syntax of writing multiple properties and values
    • One or more CSS properties and values separated by semicolons
    • Example, <div style="color:blue; text-align:center;">
  • Overrides any style set globally
    • The style attribute will override any style set globally, like styles specified in the <style> tag or in an external style sheet
    • The style attribute can be used on any HTML element
    • It will validate on any HTML element, however, it is not necessarily useful
